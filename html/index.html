Gemma Agent Orchestrator

Gemma 4 기반 로컬 에이전트 오케스트레이터 — 설정 주도, 모듈형, 로컬 우선

시스템 특징

로컬 우선

Gemma 4를 Ollama로 로컬 실행. 인터넷 연결 없이도 완전히 동작합니다.

설정 주도

모델명·툴명·경로·API 키가 코드에 없음. 전부 YAML로 조립합니다.

런타임 분리

reasoning / files / browser / api 런타임이 독립적. 하나가 죽어도 나머지는 살아있습니다.

다층 에이전트

main_agent가 researcher, coder, file_worker에게 작업을 위임합니다.

MCP 통합

YAML 한 줄로 MCP 서버를 연결. 코드 수정 불필요.

완전한 관측성

모든 단계가 JSONL 트레이스에 기록. 재생·디버그·비용 분석 가능.
