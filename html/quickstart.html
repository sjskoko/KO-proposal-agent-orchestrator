빠른 시작

설치부터 첫 번째 에이전트 실행까지

1단계 — 사전 조건 확인

항목최소 버전확인 방법
Python3.11python3 --version
Ollama최신ollama --version
RAM16GB (27b 모델 기준)시스템 정보
Node.js18 (MCP 사용 시)node --version

2단계 — Ollama & Gemma 4 설치

# Ollama 설치 (macOS / Linux)
curl -fsSL https://ollama.ai/install.sh | sh

# Gemma 4 모델 다운로드 (~16GB, 처음 한 번만)
ollama pull gemma4:27b

# RAM 부족 시 소형 모델
ollama pull gemma4:12b

# Ollama 서버 실행 (별도 터미널에서 유지)
ollama serve

3단계 — 프로젝트 설치

git clone <repo-url>
cd gemma-agent

# 자동 설정 스크립트 (venv 생성 + 패키지 + .env 복사)
bash scripts/setup_local.sh

# 가상환경 활성화
source .venv/bin/activate
수동 설치가 필요하다면 python3 -m venv .venv && source .venv/bin/activate && pip install -e ".[dev]" && cp config/env.example .env

4단계 — 환경 변수 설정

.env 파일을 열고 확인합니다. 로컬만 쓴다면 기본값 그대로도 됩니다.

# .env
OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_MODEL=gemma4:27b

# API 폴백 (선택 — 없어도 동작)
OPENAI_API_KEY=
ANTHROPIC_API_KEY=

5단계 — 첫 실행

# 단일 목표 실행
agent run "Python으로 피보나치 수열 함수 작성해줘"

# 출력 예시:
# [Planning] 목표를 2개 단계로 분해했습니다
# [Executing] step 1/2: 함수 작성 (reasoning)
# [Executing] step 2/2: 코드 검토 (reasoning)
# ╭─ Result ──────────────────────────────────╮
# │ def fibonacci(n):                         │
# │     if n <= 1: return n                   │
# │     return fibonacci(n-1) + fibonacci(n-2)│
# ╰───────────────────────────────────────────╯

6단계 — REPL 세션

agent repl

# 프롬프트
> 오늘 할 일 목록 3개 작성해줘
  → (Gemma 4가 응답)

> workspace/notes.txt 파일 읽어서 요약해줘
  → (files 런타임이 파일 읽고 요약)

> /quit

7단계 — 단위 테스트로 설치 확인

# Ollama 없이도 실행 가능
bash scripts/run_tests.sh unit

# 예상 출력:
# tests/unit/test_permissions.py ....
# tests/unit/test_tool_registry.py .....
# tests/unit/test_model_router.py .....
# 14 passed in 0.42s
다음 단계 에이전트를 커스터마이즈하려면 에이전트 설정을, 새 스킬을 만들려면 스킬 개발을 참고하세요.
