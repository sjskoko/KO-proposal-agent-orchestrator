MCP 서버 연결

Model Context Protocol 서버를 설정 파일 한 줄로 연결하는 방법

MCP란? Model Context Protocol은 에이전트가 외부 도구(파일시스템, 데이터베이스, API 등)를 표준화된 방식으로 호출하는 프로토콜입니다. 이 시스템은 stdio 트랜스포트를 지원합니다.

내장 MCP 설정 파일

파일서버기본 상태필요 조건
filesystem_server.yaml@modelcontextprotocol/server-filesystem활성Node.js
brave_search.yaml@modelcontextprotocol/server-brave-search비활성BRAVE_API_KEY

Filesystem MCP 서버 활성화

# mcp/configs/filesystem_server.yaml — 이미 활성화 상태
id: mcp_filesystem
transport: stdio
command: npx
args:
  - "-y"
  - "@modelcontextprotocol/server-filesystem"
  - "./workspace"         # 접근 허용 디렉토리
capabilities:
  - read_file
  - write_file
  - list_directory
permissions_required:
  - file_read
  - file_write
enabled: true             # ← 이미 true

Brave Search MCP 서버 활성화

# 1. .env에 API 키 추가
BRAVE_API_KEY=BSA...

# 2. mcp/configs/brave_search.yaml 수정
enabled: true   # false → true

# 3. 에이전트에 검색 툴 허용
# agents/sub_agents/researcher.yaml
tools:
  - mcp_brave_search__web_search
permissions:
  network_access: true
  api_call: true

커스텀 MCP 서버 추가

1. 설정 파일 작성

# mcp/configs/my_server.yaml
id: mcp_my_server
transport: stdio
command: python
args:
  - "-m"
  - "mcp.servers.my_server"

# 환경 변수 주입 (선택)
env:
  MY_API_KEY: "${MY_API_KEY}"

capabilities:
  - my_tool_a
  - my_tool_b

permissions_required:
  - api_call

enabled: true
description: "내 커스텀 MCP 서버 설명"

2. 서버 구현

# mcp/servers/my_server/__init__.py  (비워두기)

# mcp/servers/my_server/server.py
from mcp.server import Server
from mcp.server.models import InitializationOptions
import mcp.types as types

app = Server("my_server")

@app.list_tools()
async def list_tools() -> list[types.Tool]:
    return [
        types.Tool(
            name="my_tool_a",
            description="첫 번째 도구 설명",
            inputSchema={
                "type": "object",
                "required": ["param1"],
                "properties": {
                    "param1": {"type": "string", "description": "첫 번째 파라미터"}
                }
            }
        )
    ]

@app.call_tool()
async def call_tool(name: str, arguments: dict) -> list[types.TextContent]:
    if name == "my_tool_a":
        result = f"param1 = {arguments['param1']}"
        return [types.TextContent(type="text", text=result)]
    raise ValueError(f"Unknown tool: {name}")

if __name__ == "__main__":
    import asyncio
    from mcp.server.stdio import stdio_server

    async def main():
        async with stdio_server() as (read_stream, write_stream):
            await app.run(read_stream, write_stream,
                          InitializationOptions(server_name="my_server"))
    asyncio.run(main())

3. 에이전트에 연결

# agents/sub_agents/my_agent.yaml
tools:
  - mcp_my_server__my_tool_a   # 형식: mcp_{서버ID}__{툴이름}

permissions:
  mcp_call: true
  api_call: true               # 서버가 api_call 권한을 요구하는 경우

MCP 도구 이름 규칙

에이전트의 tools 목록에서 MCP 도구를 참조할 때:

# 형식: mcp_{설정파일의 id}__{MCP 서버의 툴이름}
mcp_filesystem__read_file
mcp_filesystem__write_file
mcp_brave_search__web_search
mcp_my_server__my_tool_a
자동 등록 부팅 시 mcp/configs/의 모든 YAML을 스캔합니다. enabled: true인 서버가 시작되고, 해당 도구들이 ToolRegistry에 자동 등록됩니다.
