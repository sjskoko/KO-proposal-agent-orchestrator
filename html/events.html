이벤트 & 트레이스

모든 실행 단계를 기록하고, 재생하고, 디버그하는 방법

이벤트 타입

이벤트발생 시점주요 필드
AgentStartedEvent에이전트 실행 시작agent_id, session_id, goal
AgentDoneEvent에이전트 실행 완료success, summary
TaskDispatchedEvent노드 실행 직전task_id, task_type, runtime_id
ToolCalledEvent툴 호출 완료tool_id, inputs, outputs, latency_ms, success
ModelQueriedEvent모델 응답 완료provider, model, input_tokens, output_tokens, latency_ms
StepFailedEvent노드 실행 실패task_id, error_type, error_message, retriable
PermissionDeniedEvent권한 거부capability, requested_by

트레이스 파일

각 세션은 data/traces/<session_id>.jsonl에 기록됩니다.

# 파일 예시 (한 줄 = 한 이벤트)
{"_type": "AgentStartedEvent", "agent_id": "main_agent", "session_id": "a1b2c3d4",
 "goal": "workspace 파일 요약", "timestamp": "2026-05-06T10:00:00.123456+00:00"}

{"_type": "TaskDispatchedEvent", "task_id": "f3a1", "runtime_id": "files",
 "task_type": "file_read", "timestamp": "2026-05-06T10:00:00.456789+00:00"}

{"_type": "ToolCalledEvent", "tool_id": "read_file", "success": true,
 "latency_ms": 12.4, "outputs": {"content": "...", "path": "./workspace/a.txt"},
 "timestamp": "2026-05-06T10:00:00.789000+00:00"}

{"_type": "ModelQueriedEvent", "provider": "gemma4_local", "model": "gemma4:27b",
 "input_tokens": 512, "output_tokens": 128, "latency_ms": 1843.2,
 "timestamp": "2026-05-06T10:00:02.632000+00:00"}

{"_type": "AgentDoneEvent", "success": true, "summary": "3개 파일 요약 완료",
 "timestamp": "2026-05-06T10:00:02.900000+00:00"}

CLI로 트레이스 조회

# 특정 세션 트레이스 출력
agent replay data/traces/a1b2c3d4.jsonl

# 출력:
# 001 AgentStartedEvent     agent=main_agent  goal=workspace 파일 요약
# 002 TaskDispatchedEvent   task=f3a1  runtime=files
# 003 ToolCalledEvent       tool=read_file  latency=12.4ms  ✓
# 004 ModelQueriedEvent     provider=gemma4_local  tokens=640  latency=1843ms
# 005 AgentDoneEvent        success=True

Python에서 트레이스 분석

from core.events.trace import TraceWriter

records = TraceWriter.replay("data/traces/a1b2c3d4.jsonl")

# 모델 토큰 사용량 집계
total_tokens = sum(
    r.get("input_tokens", 0) + r.get("output_tokens", 0)
    for r in records if r["_type"] == "ModelQueriedEvent"
)
print(f"총 사용 토큰: {total_tokens}")

# 실패한 스텝 조회
failures = [r for r in records if r["_type"] == "StepFailedEvent"]
for f in failures:
    print(f"실패: {f['task_id']} — {f['error_message']}")

커스텀 이벤트 핸들러 등록

from core.events.bus import EventBus
from core.events.types import ModelQueriedEvent

bus = EventBus()

# 특정 이벤트 타입만 구독
def on_model_query(event: ModelQueriedEvent):
    cost = (event.input_tokens / 1_000_000 * 0.50 +
            event.output_tokens / 1_000_000 * 1.50)
    print(f"[비용] {cost:.4f}$ — {event.provider}")

bus.subscribe(ModelQueriedEvent, on_model_query)

# 모든 이벤트 구독
bus.subscribe_all(lambda e: print(type(e).__name__))
