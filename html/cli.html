CLI 레퍼런스

agent 커맨드의 모든 옵션

전역 사용법

agent [COMMAND] [OPTIONS]
# 도움말
agent --help
agent run --help

agent run

단일 목표를 실행하고 결과를 출력한 뒤 종료합니다.

agent run [OPTIONS] GOAL
옵션기본값설명
GOAL필수에이전트에게 전달할 목표 문자열
--agent, -amain_agent사용할 에이전트 ID
--config, -cconfig/설정 디렉토리 경로
# 예시
agent run "Python 피보나치 함수 작성해줘"
agent run "workspace/data.csv 분석해줘" --agent coder
agent run "최신 뉴스 검색해줘" -a researcher -c /path/to/config

agent repl

인터랙티브 대화 세션을 시작합니다. 한 세션 동안 메모리가 유지됩니다.

agent repl [OPTIONS]
옵션기본값설명
--agent, -amain_agent사용할 에이전트 ID
--config, -cconfig/설정 디렉토리 경로
# 예시
agent repl
agent repl --agent researcher

# REPL 내부
> 안녕하세요
> workspace/notes.txt 파일 읽어서 요약해줘
> /quit   ← 종료 (exit, quit도 동작)

agent replay

저장된 트레이스 파일을 읽어 실행 기록을 출력합니다.

agent replay TRACE_FILE
인수설명
TRACE_FILEdata/traces/*.jsonl 파일 경로
# 예시
agent replay data/traces/a1b2c3d4.jsonl

# 출력
001 AgentStartedEvent     agent_id=main_agent
002 TaskDispatchedEvent   task_id=f3a1  runtime_id=files
003 ToolCalledEvent       tool_id=read_file  latency_ms=12.4
004 ModelQueriedEvent     provider=gemma4_local  tokens=640
005 AgentDoneEvent        success=True

setup_local.sh

# - Python venv 생성

# - .env 파일 복사

run_agent.sh

bash scripts/run_agent.sh "목표 문자열" [agent_id]

bash scripts/run_agent.sh "코드 리뷰해줘" coder

bash scripts/run_tests.sh [all|unit|integration|agent|skill]

bash scripts/run_tests.sh all           # 전체
